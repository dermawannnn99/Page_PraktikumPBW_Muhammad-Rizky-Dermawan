WHAT I LEARNED

Come to think of it, when I first joined this Web Programming class, I thought building a website was just about added text, mixing colors and placing images. But it turns out, the journey goes much deeper than that. It all started in the classroom, where the instructor laid a solid theoretical foundation, from how the internet works to the logic behind well-structured code. That’s when I realized that behind a simple web interface lies an architecture that must be meticulously constructed. What’s exciting is that the theory doesn’t just stay on paper. During lab sessions, the lab assistants were incredibly helpful in diving straight into the world of coding. Under their guidance, I practiced what the professor taught. Every weekly exercise and assignment truly challenged my mind. From initially only being able to create static text with HTML and simple CSS styling, I gradually began to build “dynamic” websites using JavaScript logic. The highlight came when studying PHP and databases; I gained an understanding of how to manage data on the server and build functional systems. This experience opened my eyes to the fact that web development isn’t just about typing code, but about the art of systematically solving problems.
